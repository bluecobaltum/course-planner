# 课程数据导入模板

> 按此模板填写课程数据后发给我，我会自动转换为 `data/courses.json` 并跑测试验证。

---

## 一、字段说明

| 字段 | 类型 | 必填 | 说明 | 示例 |
|------|------|------|------|------|
| `section_id` | 字符串 | ✅ | 教学班唯一 ID，格式 `课程代码-班号` | `MATH101-01` |
| `course_code` | 字符串 | ✅ | 课程代码，同一门课的不同班共享 | `MATH101` |
| `course_name` | 字符串 | ✅ | 课程中文名 | `高等数学A` |
| `credit_transfer_group` | 字符串 | ✅ | 学分互认组，**同一组内只能选一个班** | `MATH101` |
| `credits` | 整数 | ✅ | 学分数 (1-10) | `4` |
| `teacher.name` | 字符串 | ✅ | 教师姓名 | `王教授` |
| `teacher.rating` | 小数 | ✅ | 学生评教分数 **(1.0~5.0)** | `4.5` |
| `schedule` | 数组 | ✅ | 每周上课时间 | 见下方说明 |
| `schedule[].day` | 整数 | ✅ | 星期几 **(1=周一 ~ 5=周五)** | `1` |
| `schedule[].start` | 整数 | ✅ | 开始节次 **(1-14)** | `3` |
| `schedule[].end` | 整数 | ✅ | 结束节次 **(1-14)**，≥start | `4` |
| `location.building` | 字符串 | ✅ | 教学楼名称 | `汇文楼` |
| `location.floor` | 整数 | ✅ | 楼层 (线上填 0) | `3` |
| `location.room` | 字符串 | ✅ | 教室号 (线上填 `-`) | `301` |
| `course_type` | 字符串 | ✅ | **`major`** = 专业课/必修 | **`easy`** = 水课/选修 |
| `delivery_mode` | 字符串 | ✅ | `线下传统` / `线上网课` / `线上线下混合` |
| `semester` | 字符串 | ❌ | 学期标识，默认 `2025-2026-2` | `2025-2026-2` |

## 二、节次对照表

| 节次 | 时段 | 俗称 |
|------|------|------|
| **第 1-5 节** | 上午 | — |
| 第 1-2 节 | 上午最早 | **早八** ← 评分系统重点关注的 |
| 第 3-5 节 | 上午后半 | — |
| **第 6-10 节** | 下午 | — |
| 第 6-8 节 | 下午前半 | — |
| 第 9-10 节 | 下午后半 | — |
| **第 11-14 节** | 晚上 | — |

> 共 **1-14 节**。上午=1-5，下午=6-10，晚上=11-14。
> `schedule` 里填节次数字，不要填钟点。比如"周一 10:00-11:40" → `{"day": 1, "start": 3, "end": 4}`

## 四、时间写法规范

统一用"周X第Y-Z节"格式，多个时段用逗号分隔：

```
✅ 正确写法：
   周一1-2节, 周三1-2节, 周五1-2节
   周二3-4节, 周四3-4节
   周三9-10节
   无固定时间（异步网课）        ← 线上异步课，schedule 留空

❌ 错误写法：
   周一早上8点                   ← 不要用钟点
   Mon 10:00-11:40              ← 不要用英文
   周一第1-2节，周三第1-2节       ← 用英文逗号不是中文逗号
```

## 五、评分策略提示（制作数据时参考）

为了让 Demo 有故事可讲，建议制造以下冲突：

| 冲突类型 | 做法 | Demo 效果 |
|---------|------|----------|
| 高分教授在早八 | 某课评分最高的班放在 1-2 节 | `gpa_focus` 选它，`no_morning` 不选 |
| 水课时间冲突 | 热门水课和必修课撞时间 | 产生"换一门水课"的方案差异 |
| 网课选项 | 给某门 major 课加一个线上班（评分略低） | `easy_mode` 倾向选，省出时间 |
| 周五课 | 某体育课只有周五下午有 | `long_weekend` 宁可降分选另一班 |
| 教师分差 | 同门课两个教师评分差距 >0.5 | `gpa_focus` 方案差异显著 |
